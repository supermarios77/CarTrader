# CarTrader Platform

This repository contains the code and infrastructure for the CarTrader marketplace platform.

## Structure

- `apps/` – Application services (frontend, API gateway, domain services).
- `packages/` – Shared libraries, configuration, and tooling.
- `infra/` – Infrastructure as code, deployment manifests, container assets.
- `docs/` – Architecture decision records, onboarding guides, API documentation.
- `ops/` – Operational playbooks, runbooks, and metrics definitions.
- `tools/` – Internal CLIs, automation scripts, and developer tooling.
- `tests/` – Cross-service end-to-end, load, and contract test suites.

## Getting Started

This project uses Node.js 20+, PNPM, and Turborepo. Detailed setup instructions will be added as the services are implemented.
